#混合现实：把真实世界叠加进虚拟世界#

本文摘自前哨王煜全第二季第一期

在三大现实（VR，AR，MR）中，MR好像是知名度最低的。
混合现实相对于AR把虚拟的东西叠加到真实的世界，混合现实则是把真实的东西叠加到虚拟的世界。看起来差不多，但其实MR难度大多了。因为首先得把现实的东西虚拟化，也就是先得用摄像头捕捉画面，但我们都知道摄像头捕捉的画面都是二维的，也就是画面是扁平的，没有立体感，所以还得把二维的图像通过计算机形成三位的虚拟图像，这叫3D建模，只有这样虚拟化之后，才能把它很好的融合进虚拟的3D世界里面。
2017年6月6日，苹果召开了全球开发者大会。在这个大会里，有个不为大家所重视的东西。“增强现实平台”(ARkit)。在其演示的时候，明明只有个桌面，但在手机屏幕里，桌面上就出现了一个被子，一盏台灯。这个好像和Pokemon Go 没有什么区别。但很多人没有注意到，苹果在增强现实平台的展示中，随意挪动真实桌面上的虚拟台灯，虚拟茶杯，它们的影子其实也会随之变化，这就要求摄像头对真实的桌面进行实时的3D建模，说明苹果平台中已经具备摄像头的3D建模能力了。

##罗辑思维笔记##

  1. 毕昇和德国人古滕堡各自独立发明了活字印刷术，毕昇虽然比古滕堡早了400年，但他发明之后，只有非常小规模的实验，然后就很少用了。因为那个时代烧不出整齐的胶泥活字，印出来的字都对不齐。还有强度不够，印不了几张就毁了。一直到明清，中国印刷的主流仍然是雕版印刷术。后来要不是为了和西方人比贡献，才在《梦溪笔谈》里把毕昇找到，否则，恐怕毕昇永远默默无闻。 而古德堡发明后只用了20年，就在欧洲迅速传播开来。—-321期 发明的基础。

  2. 纸上谈兵这个故事说的是赵括，但战国的时候没有纸，纸到东汉之后才有，再去查这个成语，都说出自史记，但史记里并没有这个词。甚至有人说，清代之前的中国典籍中都没有这个词。“纸上谈兵”其实是现代人造的一个词

  3. 话说心理学上有个词叫认知闭合需求，他是指人类受不了一个问题没答案 。比如说古代人不知道为啥下雨，那这个问题就没闭合，让人很难受，所以就用雷公电母胡说一通，满足人闭合需求。那现在互联网让我们突然面对大量的陌生问题使我们这代人的认知闭合需求也非常大。我们应该主动不闭合这个认知，这个在心理上的负担很重哦，但是我们认知世界有多了一点线索，放纵自己的认知闭合需求是这个时代特别容易犯的毛病，也是一个人变蠢的最好方法。
  4. 话说有个佛教徒家里有不少老鼠，蟑螂之类的害虫，但佛教徒的教义是不许杀生的，所以他陷入了两难。他就去向一个法师求助，现在这个难题是法师的了。如果法师说不能杀，这显得太不近人情，你总不能要求别人和老鼠蟑螂在一起忍者生活吧？说杀呢，有违背了佛祖的戒律。
    那法师是怎么说的呢?他说：家里还是卫生一代你对人比较好。你看，这个回答里有两个巧妙，第一。给出了具体的建议，说了真心话不回避问题。第二即使是真心话，也可以说的非常圆融，不显得杀气腾腾。
    这其实是我们经常遇到的一种考研，别人请教自己一个两难的问题，不仅仅是得到你的建议个结论。最重要的，还有你看待这个问题的独特的温度和智慧。

比特币简介

发表于 2017-10-08 | 分类于 杂谈

##比特币简介##

本文摘自硅谷来信第一季307期

  • 比特币是带有一个特定随机数字的区块链。这个随机数字（密钥）就是它的特征，由于大家看不见这个随机数，因此它也被称为“私钥”。
  • 如果你得到了比特币，你就拿到了一个相应的你自己看得见的密钥，也成为“公开密钥”，你就获得了这个比特币的所有权和使用权。
  • 如果你将它给了别人，你的公开密钥就作废了，比特币的这一套算法就会产生一个新的公开密钥给对方。这样比特币就完成了拥有者的转变。当然，比特币协议需要通知所有人这个币的主任变了（即旧的公钥作废，新的公钥生效）。
    还有两个问题
    1.谁来记录比特币所有权的账本呢？又是谁来判断某个人手上拿着的比特币的公钥和某个实际存在的比特币的私钥完全匹配呢？
    答案是互联网上运行比特币协议的全部计算机。
    2.这些计算机由谁来提供呢?
    答案是任何在互联网上愿意将自己的计算机贡献出来的人。当然这些人也不是白白做贡献，当他们的计算机为比特币的交易提供一些计算之后，他们就被奖励一个比特币，俗称“挖矿”。

如何在VPS上Tor

  1. 首先要有个ss服务器，然后在你的VPS上

    apt-get install tor proxychains

    ​

  2. 然后再开一个ss进程

    proxychains ssserver -c /etc/shadowsocks.json -d start

这样就可以了，ss就自然通过Tor了，但速度有点慢，因为所有经过tor访问的数据都得“三连跳”经过三次数据代理。

  1. 如何关掉呢，只需要杀死这个进程，然后再打开ss进程即可

    ssserver -c /etc/shadowsocks.json -d start

如何练口语

通过张开嘴巴练听力，通过多写东西练阅读，通过听力输入来练口语，通过大量阅读来练写作。

听和说是对应的，写和读是对应的。

想练好口语，有两个必须要练好，第一个是说话的风格，即语调，口语，语流等，这个只能通过听来实现，最好听某个人的。

第二个更重要的是我们要言之有物，这个要通过写来实现。不管我们口语多差，我们都可以说出good morning等，因为这些我们也可以轻松的写下来，当我们能轻松写下来的东西越多时，我们的口语能表达的范围也就增加了。写作的范围大于口语的范围。

控制系统的3种变量类型

  1. 用户定义变量：由用户定义名称和类型的变量，例如，计算参数

强调句与定语从句辨析

原则

如果将it is/was 与连接词去掉，句子仍正确为强调句，选用that做为连接词；否则为定语从句。

例句

（1）It is on the island that they spent 30 years.

该句为强调句。表示地点的名字the island前有介词与其连用，说明其为完整的地点状语。

去掉It is 意义仍然完整。

（2） It is the island where (on which) they spent 30 years.

该句为定语从句。表示地点的名词the island 前没有介词与其连用，说明它不是一个完整的地点状语。

如果it is去掉，意义不完整。故应选用关系副词where或on which构成定语从句结构。
