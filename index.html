如何在VPS上Tor

  1. 首先要有个ss服务器，然后在你的VPS上

    apt-get install tor proxychains

    ​

  2. 然后再开一个ss进程

    proxychains ssserver -c /etc/shadowsocks.json -d start

这样就可以了，ss就自然通过Tor了，但速度有点慢，因为所有经过tor访问的数据都得“三连跳”经过三次数据代理。

  1. 如何关掉呢，只需要杀死这个进程，然后再打开ss进程即可

    ssserver -c /etc/shadowsocks.json -d start

如何练口语

通过张开嘴巴练听力，通过多写东西练阅读，通过听力输入来练口语，通过大量阅读来练写作。

听和说是对应的，写和读是对应的。

想练好口语，有两个必须要练好，第一个是说话的风格，即语调，口语，语流等，这个只能通过听来实现，最好听某个人的。

第二个更重要的是我们要言之有物，这个要通过写来实现。不管我们口语多差，我们都可以说出good morning等，因为这些我们也可以轻松的写下来，当我们能轻松写下来的东西越多时，我们的口语能表达的范围也就增加了。写作的范围大于口语的范围。

控制系统的3种变量类型

  1. 用户定义变量：由用户定义名称和类型的变量，例如，计算参数

  2. 计算参数：专门的，预定义的计算变量，给定地址R及随后的数字，预定义的计算变量类型为REAL

  3. 系统变量：供控制系统使用的变量，它们可以在程序中进行处理，即读/写操作。系统变量名称总是以“$”符号开始，紧接着是专门的名词。

    | $S | 设定数据 |
    | $T | 刀具管理参数 |
    | $P | 程序数值 |
    | $A | 实际数值 |
    | $V | 服务参数 |
    | N | NCK全局 |
    | C | 通道专用 |

虚拟语气

将来

主would (could,might) do + 从if…. were to do/should do

例：If I should win the lottery, I would buy a house.

​ 万一我赢了彩票，我就买栋房子。

现在

主would(could,might) do + 从if…. did/were

例：If I were a millionaire, I would buy everything I want.

​ 如果我要是百万富翁，我会想买什么就买。

过去

主would(could,might) have done + 从If….had done/have been

例：I don’t know what I would have done if he hadn’t helped me.

​ 如果他当时没有伸出援手，我真不知道会怎样

Where are you from VS where do you come from

Where do you come from?

这个是不建议这么用的，这个在外国人看来是有点怪的，从哪来是过去的动作，应该用did，所以正确的表达式“Where did you come from”

它的意思是where were you immediately before you came to this location we are at right now?

并列连词

发表于 2017-10-05 | 分类于 英语

#并列连词#

简单句1+并列连词+简单句2

并列连词

  • and: QiGe is so beautiful and the Prince wants to marry her.

  • but: QiGe is so beautiful but she has bad temper.

  • for: QiGe is famous, for she has a beautiful voice.

  • nor: He won’t wait for you, nor for me.

  • or: Grab QiGe, or you’ll regret.

  • so:It’s hard for QiGe to take care for herself, so Yifan wants to help her.

  • and so: It’s hard for QiGe to take care for herself, and so Yifan wants to help her.

  • yet: The question was so simple, yet Yifan couldn’t answer it.

  • and yet:The question was so simple,and yet Yifan couldn’t answer it.

    ​

特殊情况：常见的伪连词

  • however
  • therefore
  • moreover/furthermore
  • then
  • notwithstanding
  • nevertheless

错误用法

句子1，however，句子2

正确用法

  • 句子1. However, 句子2
  • 句子1; however, 句子2
  • 句子1. 句子2第一部分, however, 句子2第二部分
  • 句子1. 句子2, however.

代词it和代词that的区分

it 特指名词及修饰语，而that仅仅指代名词本身

例子：

The weather in Shanghai is wetter than that in Beijing.(对)

The weather in Shanghai is wetter than it in Beijing.(错)
