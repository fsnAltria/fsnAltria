西门子数控变量

两个特殊的变量类型

类型 含义 值范围
AXIS 仅为轴名称（轴地址） 所有在通道中出现的轴名
FRAME 位移、旋转、缩放、镜像的几何数据

控制系统的3种变量类型

  1. 用户定义变量：由用户定义名称和类型的变量，例如，计算参数

  2. 计算参数：专门的，预定义的计算变量，给定地址R及随后的数字，预定义的计算变量类型为REAL

  3. 系统变量：供控制系统使用的变量，它们可以在程序中进行处理，即读/写操作。系统变量名称总是以“$”符号开始，紧接着是专门的名词。

    | 字母 | 含义 |
    | :–: | :—-: |
    | $M | 机床数据 |
    | $S | 设定数据 |
    | $T | 刀具管理参数 |
    | $P | 程序数值 |
    | $A | 实际数值 |
    | $V | 服务参数 |
    | N | NCK全局 |
    | C | 通道专用 |
    | A | 轴专用 |
